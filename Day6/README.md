# 📘 Day 6 - Data Analysis with NumPy & Pandas

> **Date:** 15-Jul-2026

## 📌 Overview

Day 6 focused on building practical **Data Analysis** skills using **NumPy** and **Pandas**, the two core Python libraries for working with numerical and tabular data.

Throughout this day, I learned how to create and manipulate **NumPy arrays**, understood the concept of **Broadcasting**, worked extensively with **Pandas DataFrames**, learned to read data from multiple file formats (**CSV, Excel, JSON**), and practiced essential data preparation techniques such as **filtering, grouping, handling missing values, preprocessing, and basic feature engineering**.
