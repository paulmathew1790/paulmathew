# 📘 Day 8 - Machine Learning Fundamentals

> **Date:** 20-Jul-2026

## 📌 Overview

Day 8 focused on the **Fundamentals of Machine Learning**, covering the end-to-end ML lifecycle, the difference between supervised and unsupervised learning, and hands-on practice with **Scikit-learn**.

Throughout this day, I learned how a machine learning project moves from raw data to a trained, evaluated model, explored the core ideas behind regression, classification, and clustering, and practiced splitting data into train/test sets and evaluating model performance using standard metrics. I applied these concepts by building a **House Price Prediction** regression model using Scikit-learn.

---

## 🎯 Learning Objectives

- Understand the stages of the Machine Learning lifecycle.
- Differentiate between supervised and unsupervised learning.
- Understand regression and classification as supervised learning tasks.
- Get an overview of clustering as an unsupervised learning technique.
- Learn how and why data is split into train/test sets.
- Understand common model evaluation metrics.
- Get hands-on with Scikit-learn basics.

## 🧩 Unsupervised Learning

- Concept and definition

- Building a simple Linear Regression model
