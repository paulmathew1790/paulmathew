# 📅 Day 4 – Mathematics for Artificial Intelligence

## 📖 Overview

Day 4 focused on building the mathematical foundation required for Artificial Intelligence and Machine Learning. The topics covered include Linear Algebra, Probability, Statistics, and introductory Calculus. Practical implementations were completed using **NumPy** in a Jupyter Notebook to strengthen conceptual understanding.
