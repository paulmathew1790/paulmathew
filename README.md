# 🚀 AI & Machine Learning Bootcamp Journey (10 Days)

Welcome to my **10-Day AI & Machine Learning Learning Journey**. This repository documents my daily progress as I learn the foundations of Artificial Intelligence, Python programming, Data Structures & Algorithms, Mathematics for AI, Data Analysis, Machine Learning, Generative AI, and MLOps.

Each day contains notes, code examples, assignments, mini-projects, Jupyter notebooks, and learning summaries.

---

## 🎯 Objectives

- Learn Python from beginner to advanced
- Build a strong foundation in Data Structures & Algorithms
- Understand the mathematics behind Machine Learning
- Perform data analysis and visualization
- Learn SQL fundamentals
- Build Machine Learning models using Scikit-learn
- Explore Generative AI, LLMs, Prompt Engineering, and RAG
- Deploy ML models using FastAPI and Docker
- Practice Git and GitHub for version control
