# 📘 Day 7 - Data Visualization & SQL

> **Date:** 16-Jul-2026

## 📌 Overview

Day 7 focused on **Data Visualization** using **Matplotlib** and the fundamentals of **SQL** for querying and managing structured data.

Throughout this day, I learned how to create basic charts, histograms, and scatter plots to visualize data patterns, explored the concept of correlation heatmaps, and built a foundation in **SQL basics** including `SELECT`, `WHERE`, `GROUP BY`, and `JOIN`. I also learned how to connect Python with databases to run SQL queries programmatically.
